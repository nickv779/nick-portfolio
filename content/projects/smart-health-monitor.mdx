## Overview

Developed for the UF EMBS Design-a-thon in October 2025, the goal was to create an all-in-one interface for predicting a user's emotions. The conceptual idea for this project was to serve as a tool for doctors to check on their patients and understand their emotional wellbeing, especially for patients whose worsened health are impacting their mental health. The project ultimately won 2nd place after an in-person judging session with judges from the IEEE Gainesville Chapter.

## Features

- Face detection model using dlib landmarks and a random forest classifier for detecting facial emotions.
- Random forest classifier trained on physiological sensor data for determining emotion, trained on a dataset from a marketing study.
- All-in-one interface developed with HTML and Tailwind CSS with Flask for connecting the models to the web interface.
- Webcam feed configured with WebRTC is passed to face detection model while physiological data is simulated by selecting a random test sample from an extra dataset of the marketing study.
